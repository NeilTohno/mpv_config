注意：
60Hz 刷新率看 23.976 帧 等视频会出现卡顿问题，
需要自己用 Custom Resolution Utility 创建 23Hz、24Hz、25Hz（59Hz 可以不用，额）；
或者直接切换到 120Hz，当然前提是有高刷新率显示器。
